hexo简介

Hexo

Hexo 是由Node.js编写的静态Blog框架。它主要涉及：Node.js, stylus, Markdown 等东东。和作者@tommy351说的一样，整个框架的应用，主题转换与修改都是非常简单，方便的。 个人选择 Hexo 的理由

  • 在Github Page上应用简单
  • 主题自己可以修改, 本博主题 pacmanBlue
  • Markdown
